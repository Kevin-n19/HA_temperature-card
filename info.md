# Temperature Card

A custom card for Home Assistant to display temperature or humidity with battery levels and last update time.

## Features

- 📊 Display temperature or humidity values
- 🔋 Optional battery level indicator with low battery warning
- 🕒 Last update time with stale data alert
- 🎨 Color-coded values based on configurable thresholds
- 🖱️ Click to open more-info dialog

## Preview

The card displays:
- Main sensor value with customizable unit
- Icon (customizable, default: thermometer)
- Battery level (optional)
- Time since last update
- Visual alerts for low battery and stale data
- Color coding for high/low temperature thresholds
